=====Help File for 716 DemoBot main code=====

-h, --help, help	Displays this help page

-v, --verbose		Enables More Detailed logging printed in bash

-w, --wireless		Switches Control Schemes to correspond with Xbox One wireless controls and enables controller connectivity checking via the l2ping utility

--test-interfaces	Checks and reports connectivity status with the PWM driver HAT before launching main loop (Cannot Currently be Used With -w)
