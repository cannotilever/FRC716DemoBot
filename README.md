# FRC716DemoBot
## CURRENT BUILD HAS NOT BEEN TESTED ##
Repo for Python Control code for the Raspberry Pi driven Demonstration Robot.
This Code is Still in Progress as of 11/26/19 but is suitable for production.

Designed Loadout:
2 CIM tankdrive drivetrain
Green ball launcher
	Magazine Gen 1
	NO turntable
	NO Gen 2 Magazine
1 Pi
	PWM driver HAT
Xbox One wireless controller
	Optional USB wired support
	Automatic Emergency Stop on controller latency above 60ms
Future loadouts in the pipeline
More Pis
	Sense HAT for auto mode navegation
	PiFace HAT for LED control and Digital inputs
Saftey
	Emergency STOP on accelerometer detected impact
