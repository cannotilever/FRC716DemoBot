# FRC716DemoBot

Repo for Python Control code for the Raspberry Pi driven Demonstration Robot.
This Code is Still in Progress as of 11/26/19 but is suitable for production.

## Designed Loadout: ##
1. 2 CIM tankdrive drivetrain
1. Green ball launcher
	1. Magazine Gen 1
	1. NO turntable
	1. NO Gen 2 Magazine
1. 1 Pi 3b+
	1. PWM driver HAT
1. Xbox One wireless controller
	1. Optional USB wired support
	1. Automatic Emergency Stop on controller latency above 60ms
## Future possible loadouts: ##
1. More Pi's
	1. Sense HAT for auto mode navegation
	1. PiFace HAT for LED control and Digital inputs
1. Saftey
	1. Emergency STOP on accelerometer detected impact
